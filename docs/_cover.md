# Pict Section OpenSeaDragon

> A deep-zoom image viewer and annotation section for Pict applications

Wraps OpenSeaDragon and Annotorious as a first-class Pict view. Drop it into any Pict application to get zoomable imagery, W3C Web Annotations, a configurable color palette, drawing tools, hatch fills, and a linked comments sidebar -- all driven by a single configuration object.

- **Deep-Zoom** -- OpenSeaDragon tile sources, zoom / home / fullscreen controls
- **W3C Annotations** -- Annotorious editor with circle, ellipse, rect, freehand, polygon, line, and arrow tools
- **Color Palette** -- Declarative `Colors` map drives the toolbar and per-annotation stylesheets
- **Hatch Fill Mode** -- Optional diagonal hatch pattern for highlighted regions
- **Comments Panel** -- Sibling view renders a scrollable comments / tags sidebar
- **Extensible Hooks** -- Every annotation event is an override point on the view class
